##Proceso en CLI 

1. Limpiar el .jar 
```
mvn clean
```
2. construir el .jar
```
mvn package 
```
3. ejecutar el Main
```
java -cp target/PrincipiosSOLID-SolucionTarea-1.0-SNAPSHOT.jar com.mycompany
```
4. ejecutar los TEST
```
mvn TEST
